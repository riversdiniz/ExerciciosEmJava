#### Exercícios propostos:

Fazer um programa para ler quatro valores inteiros A, B, C e D. A seguir, calcule e mostre a diferença do produto
de A e B pelo produto de C e D segundo a fórmula: DIFERENCA = (A * B - C * D).

| Entrada | Saída | 
|---------|-------|
|  5     |  DIFERENCA = -26 |
|  6     |
|  7     |
|  8     |

| Entrada | Saída | 
|---------|-------|
|  5     |  DIFERENCA = 86 |
|  6     |
|  - 7     |
|  8     |
